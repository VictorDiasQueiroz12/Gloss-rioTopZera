Como criar uma NavBar?

Uma NavBar (barra de navegação) permite ao usuário navegar entre páginas ou seções.

Exemplo de código HTML:

<nav class="navbar">
  <a href="#home">Home</a>
  <a href="#sobre">Sobre</a>
  <a href="#contato">Contato</a>
</nav>

CSS para NavBar:

.navbar {
  background: #003366;
  padding: 12px 18px;
  border-radius: 10px;
  box-shadow: 0 0 18px #39ff14, 0 0 50px #39ff1433;
  display: flex;
  gap: 24px;
  justify-content: center;
}
.navbar a {
  color: #39ff14;
  font-weight: 600;
  text-decoration: none;
  padding: 6px 18px;
  border-radius: 6px;
  transition: background .2s, color .2s;
  text-shadow: 0 0 7px #39ff14cc;
  letter-spacing: .5px;
}
.navbar a:hover {
  background: #39ff14;
  color: #003366;
}

Como criar um Accordion?

Accordion é um painel expansível para esconder/mostrar conteúdo.

Exemplo de código HTML:

<details class="accordion">
  <summary>Clique para expandir</summary>
  <p>Conteúdo do accordion.</p>
</details>

CSS para Accordion:

.accordion {
  background: #01334b;
  border-radius: 8px;
  box-shadow: 0 0 12px #39ff1433;
  padding: 10px 15px;
  margin-bottom: 10px;
}
.accordion summary {
  color: #39ff14;
  cursor: pointer;
  font-weight: 600;
  font-size: 1.06em;
  text-shadow: 0 0 7px #39ff14;
}
.accordion[open] summary {
  color: #fff;
  text-shadow: none;
}
.accordion p {
  color: #fff;
  margin: 10px 0 0 15px;
}

Como criar um Card?

Card é um bloco de destaque para informações, produtos ou perfis.

Exemplo de código HTML:

<div class="card-exemplo">
  <h3>Título do Card</h3>
  <p>Descrição do card.</p>
</div>

CSS para Card:

.card-exemplo {
  background: #04669a;
  border-radius: 10px;
  box-shadow: 0 0 16px #39ff1440;
  padding: 16px 22px;
  color: #fff;
  width: 210px;
  margin: 0 auto;
}
.card-exemplo h3 {
  color: #39ff14;
  margin-top: 0;
  text-shadow: 0 0 7px #39ff1466;
}

Como criar um Botão de voltar à página?

Um botão que retorna para a página anterior usando JavaScript.

Exemplo de código HTML:

<button class="btn-voltar" onclick="window.history 
.back()">Voltar</button>

CSS para botão:

.btn-voltar {
  color: #39ff14;
  background: #002b36;
  border: 1.5px solid #39ff14;
  border-radius: 6px;
  padding: 8px 20px;
  font-weight: 600;
  cursor: pointer;
  font-size: 1em;
  transition: background .2s, color .2s;
  text-shadow: 0 0 7px #39ff14cc;
}
.btn-voltar:hover {
  background: #39ff14;
  color: #002b36;
}
