<Glossário Interativo de Tags JS>



O que é o comando IF?

Executa um bloco de código se uma condição for verdadeira.

Exemplo:

if (idade >= 18) {
    console.log("Você é maior de idade.");
}
O que é o comando ELSE?

Executa um bloco de código caso a condição do IF seja falsa.

Exemplo:

if (idade >= 18) {
    console.log("Maior de idade.");
} else {
    console.log("Menor de idade.");
}
O que é o comando SWITCH?

Permite executar diferentes blocos de código com base no valor de uma expressão.

Exemplo:

switch (cor) {
    case "vermelho":
        console.log("A cor é vermelha.");
        break;
}
O que é o comando CASE?

Define um bloco de código dentro do SWITCH para um valor específico.

Exemplo:

switch (dia) {
    case "segunda":
        console.log("Início da semana.");
        break;
    case "sexta":
        console.log("Quase fim de semana!");
        break;
}
