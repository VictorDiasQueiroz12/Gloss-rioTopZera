<Glossário Interativo de Propriedades CSS>



@import

Usado para importar arquivos CSS externos, como fontes do Google Fonts.

Exemplo:

@import url('https://fonts.googleapis.com/css2?
family=Roboto');
margin

Define o espaço externo ao redor de um elemento.

Exemplo:

margin: 20px;
padding

Define o espaço interno entre o conteúdo do elemento e sua borda.

Exemplo:

padding: 16px;
background

Define a cor ou imagem de fundo de um elemento.

Exemplo:

background: #0D0D0D;
background-image

Define uma imagem como fundo de um elemento.

Exemplo:

background-image: url(fundo.png);
background-size

Controla o tamanho da imagem de fundo.

Exemplo:

background-size: cover;
background-position

Posiciona a imagem de fundo dentro do elemento.

Exemplo:

background-position: center;
background-repeat

Define se a imagem de fundo será repetida.

Exemplo:

background-repeat: no-repeat;
color

Define a cor do texto de um elemento.

Exemplo:

color: #FFFFFF;
font-family

Define a família de fontes para o texto.

Exemplo:

font-family: 'Roboto', sans-serif;
font-size

Define o tamanho da fonte.

Exemplo:

font-size: 1.5em;
font-weight

Define a espessura da fonte.

Exemplo:

font-weight: bold;
border-radius

Arredonda os cantos de um elemento.

Exemplo:

border-radius: 16px;
box-shadow

Aplica uma sombra ao redor do elemento.

Exemplo:

box-shadow: 0 2px 12px rgba(0,0,0,0.2);
display

Define como o elemento será exibido na tela.

Exemplo:

display: flex;
justify-content

Alinha os itens no eixo principal de um contêiner flex.

Exemplo:

justify-content: center;
align-items

Alinha os itens no eixo cruzado de um contêiner flex.

Exemplo:

align-items: flex-start;
gap

Define o espaçamento entre elementos filhos em um contêiner flex ou grid.

Exemplo:

gap: 40px;
transition

Define a duração e o tipo de transição entre estados de um elemento.

Exemplo:

transition: transform 0.3s;
